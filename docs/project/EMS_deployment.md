### 1.5 商業設備與 ipig_system 整合方式

| 整合方式 | 支援設備 | 難度 | 說明 |
|---------|---------|------|------|
| **REST API pull** | Testo Cloud、HOBOlink | 低 | 排程每 5 分鐘拉一次，寫入 `sensor_readings` |
| **MQTT 推送** | 部分 Testo 型號 | 低 | 設備主動推，即時性更好 |
| **CSV 定期匯入** | 幾乎全部 | 中 | 需排程腳本解析 CSV |
| **Modbus TCP** | 工業型號 | 中 | 需加 Modbus → HTTP 閘道 |
| **viewLinc API** | Vaisala | 中高 | REST API，或直連 MS SQL |

**推薦整合流程（Testo Cloud API）：**

```
Testo Cloud REST API
    ↓ services/scheduler.rs 每 5 分鐘排程
ipig_system 後端
    ↓
sensor_readings 表（TimescaleDB）
    ↓
Dashboard 面板 / 告警引擎（R21-6~R21-7）
```

## 三、DIY 自組方案（路線 B）— 感測器選型

### 2.1 動物房 — 溫濕度感測器

| 型號 | 溫度精度 | 濕度精度 | 介面 | 單價 | 推薦 |
|------|---------|---------|------|------|------|
| **SHT31** | ±0.3°C | ±2% RH | I2C | ~NT$150 | ⭐⭐⭐⭐⭐ 首選 |
| SHT40 | ±0.2°C | ±1.8% RH | I2C | ~NT$200 | ⭐⭐⭐⭐⭐ 更準 |
| DHT22 | ±0.5°C | ±2% RH | 單線 | ~NT$80 | ⭐⭐⭐ 偶有讀值失敗 |
| AM2302 | ±0.5°C | ±2% RH | 單線 | ~NT$100 | ⭐⭐⭐ DHT22 加強版 |

**決定：SHT31**（I2C 介面穩定、工業級、開發資源多）

### 2.2 冰箱 — 防水溫度感測器

| 型號 | 量測範圍 | 精度 | 介面 | 單價 | 推薦 |
|------|---------|------|------|------|------|
| **DS18B20（防水探針）** | -55°C ~ +125°C | ±0.5°C | 1-Wire | ~NT$80 | ⭐⭐⭐⭐⭐ 首選 |
| PT100 / PT1000 | -200°C ~ +850°C | ±0.1°C | 需 ADC 模組 | ~NT$200+ | ⭐⭐⭐ 高精度但複雜 |

**決定：DS18B20 防水版**（plug-and-play、多個可串聯同一條 1-Wire 線）

## 七、網路連線

### 方案比較

| 方案 | 優點 | 缺點 | 適用 |
|------|------|------|------|
| **WiFi 2.4GHz（推薦）** | 不用拉線、ESP32 內建 | 訊號死角、干擾 | 大多數場景 |
| 有線乙太網路 | 最穩定 | 需拉網路線 + 加 W5500 模組 | 重要設備、訊號差區域 |
| LoRa | 遠距離、省電 | 速度慢、需 Gateway | 多棟建築、戶外 |
| 4G LTE | 不依賴內網 | 月費、需 SIM 卡 | 獨立設施、偏遠地點 |

**決定：WiFi 2.4GHz**

### WiFi 部署注意事項

- 建議設一個 **IoT 專用 SSID**（隔離業務內網，避免干擾）
- 動物房有厚水泥牆時，考慮加裝 WiFi 中繼器（TP-Link RE305 ~NT$800）
- ESP32 建議設定靜態 IP，避免 DHCP 衝突

```

        │4.7kΩ
```

## 十、部署驗證步驟

1. **單節點驗證**：先買 1 個 ESP32 + 1 個 SHT31，用 Arduino IDE 寫 HTTP POST 測試，確認資料能到後端
2. **後端整合**：R21-1 ~ R21-3 完成後，確認資料寫入 `sensor_readings` 表
3. **批量部署**：單節點穩定 3 天後再購買剩餘數量
4. **告警測試**：模擬超限值（如捏住感測器使溫度上升），確認通知發送
5. **長期穩定性**：持續觀察 1 週，確認無斷線、讀值漂移問題

| 需求 | 硬體方案 |
|------|---------|
| 感測器數量 > 20 個 | 加裝 MQTT Broker（Mosquitto），對應 TODO R21-12 |

| 停電偵測 | ESP32 + UPS 模組或電池模組 |
| CO₂ 監測（動物房） | SCD40 / MH-Z19B 感測器（NT$500～800） |
| 有線備援 | ESP32 + W5500 乙太網路模組（NT$150） |
