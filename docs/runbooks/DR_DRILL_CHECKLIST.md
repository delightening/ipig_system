## 1. 演練概述

| 項目 | 說明 |
|------|------|
| **建議頻率** | 每季至少一次（或依合規要求） |
| **RTO 目標** | < 4 小時 |
| **RPO 目標** | < 1 小時 |
| **演練類型** | 備份還原驗證、或完整主機毀損模擬 |
