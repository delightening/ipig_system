# 豬博士 iPig 系統 - 使用者操作手冊 (P1-9)

> **版本：** v1.0  
> **對象：** 實驗人員、管理員、審查委員  
> **相關文件：** 專案總覽與架構 [README.md](../README.md) | 如何啟動系統 [QUICK_START.md](QUICK_START.md) | 部署與維運 [DEPLOYMENT.md](DEPLOYMENT.md)

## 1. 登入與帳號管理
- 首次登入需根據開發設定或管理員提供之初始化密碼登入。
- 登入後系統將強制要求變更密碼。

## 2. AUP 審查系統
- **提交計劃**：至「我的計劃」點擊「新增計劃」，填寫 9 大節內容。
- **審查流程**：提交後進入 IACUC 審查，審查委員可提出意見，PI 需回覆並解決意見後方可核准。

## 3. 實驗動物管理
- **動物登記**：支援 Excel 批次匯入基本資料與體重。
- **健康紀錄**：可記錄觀察計畫、手術過程與血液檢查結果。
- **狀態變更**：支援轉讓、安樂死與猝死登記。

## 4. ERP 進銷存
- **採購/銷售**：管理藥品、飼料等物資的進出庫。
- **庫存警報**：系統每日自動檢查低庫存並發送通知。
