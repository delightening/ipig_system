## 1. 登入與帳號管理

### 1.1 登入系統

1. 開啟瀏覽器前往系統網址（預設 `http://localhost:8080`）。
2. 輸入 **Email** 與 **密碼**，點擊「登入」。
3. 首次登入系統會要求強制變更密碼，設定新密碼後即可使用。

> 密碼規則：至少 8 字元，需包含大寫字母、小寫字母與數字。

### 1.2 忘記密碼

1. 於登入頁點擊「忘記密碼」。
2. 輸入註冊用的 Email。
3. 系統將寄送密碼重設連結至該信箱，連結有效期為 1 小時。

### 1.3 個人設定

路徑：點擊右上角使用者名稱 → **個人設定**

- **顯示名稱**：修改在系統中顯示的名稱。
- **變更密碼**：從側邊欄選單開啟變更密碼對話框，需輸入目前密碼。
- **語言切換**：頂部工具列的下拉選單可切換繁體中文 / English。

## 2. 儀表板

路徑：`/dashboard`

登入後的預設首頁，提供系統各項數據概覽：

- **低庫存警示**：庫存低於安全水位的品項。
- **待處理單據**：需核准的採購 / 銷貨單據。
- **統計資訊**：動物數量、進行中計畫等。

> **自訂版面**：Widget 區塊可拖曳調整位置，自訂適合自己的工作檢視。

## 3. AUP 計畫書管理

### 3.1 瀏覽計畫書列表

路徑：側邊欄 → **AUP** → **計畫書管理**

- 列表支援依「狀態」篩選（草稿、審查中、已核准等）。
- 支援即時搜尋與欄位排序。

### 3.2 新增計畫書

1. 點擊「新增計畫書」按鈕。
2. 依序填寫各節內容，系統支援自動儲存草稿。
3. 填寫完成後點擊「提交」，計畫書進入 IACUC 審查流程。

### 3.3 審查流程

| 階段 | 角色 | 動作 |
|------|------|------|
| 提交 | PI（計畫主持人） | 填寫並提交計畫書 |
| 預審 | IACUC 行政人員 | 初步審核，指派審查委員 |
| 審查 | 審查委員 / 獸醫師 | 提出審查意見 |
| 回覆 | PI | 逐條回覆審查意見 |
| 決議 | IACUC 主席 | 核准 / 要求修正 / 否決 |

### 3.4 附件管理

- 計畫書支援上傳附件（PDF、Word、Excel、圖片）。
- 上傳：在計畫書編輯頁面的「附件」區域拖曳或點擊選擇檔案。
- 刪除：僅上傳者或管理員可刪除附件。

### 3.5 變更申請（Amendment）

路徑：側邊欄 → **AUP** → **我的變更申請**

已核准的計畫書如需修改，需透過「變更申請」流程提交修改內容，經審查核准後方能生效。

## 4. 實驗動物管理

### 4.1 動物列表

路徑：側邊欄 → **動物管理** → **動物列表**

三種檢視模式：
- **列表視圖**：傳統表格，支援搜尋、品種與狀態篩選。
- **欄舍分組**：依欄舍 / 房間 / 畜欄分類顯示。
- **欄舍地圖**：視覺化的畜欄配置圖。

### 4.2 新增動物

1. 在動物列表頁面點擊「新增動物」。
2. 填寫耳號、品種、性別、出生日期、體重等資訊。
3. 支援 **Excel 批次匯入**：點擊「匯入」→ 下載範本 → 填入資料 → 上傳。

### 4.3 動物詳情頁

點擊動物耳號進入詳情頁，包含以下 Tab：

| Tab | 功能 |
|-----|------|
| **時間軸** | 所有紀錄的時間序列總覽，可快速檢視、編輯、複製、查看歷史 |
| **觀察試驗紀錄** | 新增 / 編輯觀察、異常、試驗紀錄，支援附件上傳 |
| **手術紀錄** | 記錄手術過程、生理數值、麻醉時間 |
| **體重** | 體重追蹤與新增，支援系統號開發者模式 |
| **疫苗/驅蟲** | 疫苗接種與驅蟲處理紀錄 |
| **犧牲/採樣** | 犧牲、安樂死、採樣紀錄 |
| **血液檢查** | 血液檢查結果記錄與趨勢圖 |
| **疼痛評估** | 疼痛分數評估紀錄 |
| **基本資料** | 動物基本資訊（唯讀） |
| **病理報告** | 病理組織報告上傳與管理 |
| **轉讓紀錄** | 轉讓歷史 |

### 4.4 觀察紀錄操作

1. 進入動物詳情 → **觀察試驗紀錄** Tab。
2. 點擊「新增觀察」。
3. 填寫：事件日期、紀錄性質（異常 / 試驗 / 觀察）、使用儀器、麻醉時間、內容、治療用藥。
4. 可上傳相片與文件附件。
5. 儲存後紀錄將顯示於時間軸。

> **快捷鍵**：Alt + N 可跳至下一個空白欄位，加快填寫速度。

### 4.5 獸醫師建議

獸醫師可對觀察紀錄或手術紀錄新增建議：
1. 在紀錄列表中點擊「獸醫建議」圖示。
2. 填寫建議內容，可勾選「緊急建議」。
3. 緊急建議將立即以 Email 通知計畫主持人及共同主持人。
4. 支援附件上傳（圖片、PDF、Word）。

### 4.6 批次操作

- **批次匯出**：勾選多筆動物 → 點擊「匯出」。
- **批次指派**：勾選多筆動物 → 點擊「指派到計畫書」。
- **匯入體重**：下載體重匯入範本 → 填入資料 → 上傳。

### 4.7 電子簽章

犧牲、觀察、安樂死、轉讓等紀錄支援電子簽章，符合 21 CFR Part 11 要求。簽章時需輸入密碼確認身分。

## 5. iPig ERP 進銷存

路徑：側邊欄 → **ERP**

### 5.1 基礎資料管理

| 功能 | 說明 |
|------|------|
| 商品管理 | SKU 編碼、批號追蹤、效期管理 |
| 倉庫管理 | 倉庫新增 / 編輯，倉位配置 |
| 合作夥伴 | 供應商、客戶資料維護 |

### 5.2 採購流程

1. **建立採購單 (PO)**：選擇供應商 → 新增品項 → 儲存。
2. **到貨驗收 (GRN)**：依採購單建立驗收單，確認到貨數量與品質。
3. **採購退貨 (PR)**：針對不良品建立退貨單。

### 5.3 銷貨流程

1. **建立銷貨單 (SO)**：選擇客戶 → 新增品項 → 儲存。
2. **出貨單 (DO)**：依銷貨單建立出貨單。

### 5.4 倉儲作業

- **庫存調整 (ADJ)**：手動調整庫存數量（需填寫原因）。
- **庫存盤點 (STK)**：建立盤點單，逐一確認各品項實際數量。
- **調撥 (TR)**：跨倉庫調撥庫存。
- **報廢 (RM)**：記錄過期或損壞品項的報廢。

### 5.5 庫存查詢

- **庫存現況** (`/inventory`)：查看各品項目前庫存水位。
- **庫存流水帳** (`/inventory/ledger`)：追蹤每筆庫存異動紀錄。
- **倉庫佈局** (`/inventory/layout`)：視覺化檢視倉位配置。

### 5.6 低庫存警示

系統每日自動檢查庫存水位，低於安全庫存或再訂購點時將發送通知。

## 6. 人事管理 (HR)

路徑：側邊欄 → **人員管理**

> 注意：純審查角色（審查委員、獸醫師、IACUC 主席、PI）不顯示此選單。

### 6.1 出勤打卡

路徑：**人員管理** → **出勤打卡**

- **今日打卡**：點擊「上班打卡」或「下班打卡」（可設定 IP / GPS 限制）。
- **歷史紀錄**：查看個人歷史打卡紀錄。

### 6.2 請假管理

路徑：**人員管理** → **請假管理**

1. 點擊「申請請假」。
2. 選擇假別（特休、病假、事假等）、日期區間。
3. 可上傳證明文件（如診斷證明）。
4. 送出後由主管審核。

### 6.3 加班管理

路徑：**人員管理** → **加班管理**

申請加班並記錄加班時數，送出後由主管審核。

### 6.4 特休額度管理

路徑：**人員管理** → **特休額度管理**（需 `hr.balance.manage` 權限）

管理員可查看並調整員工特休額度，支援匯出過期報表為 CSV。

### 6.5 日曆

路徑：**人員管理** → **日曆**

整合 Google Calendar，同步管理行事曆。

## 8. 系統管理（管理員專用）

路徑：側邊欄 → **系統管理**（僅 admin 角色可見）

### 8.1 使用者管理

- 新增、編輯、停用使用者帳號。
- 指派角色（可多角色）。
- 重設使用者密碼。
- **模擬登入**：admin 可暫時以其他使用者身分登入，用於除錯（畫面上方會顯示藍色提示橫幅）。

> **敏感操作保護**：刪除使用者、重設密碼、模擬登入等操作需重新輸入管理員密碼確認。

### 8.2 角色權限

管理系統角色與對應權限，內建角色包含：

| 角色 | 說明 |
|------|------|
| 系統管理員 | 全系統最高權限 |
| 計畫主持人 (PI) | 提交計畫、管理動物 |
| 獸醫師 (VET) | 審查計畫、動物健康管理 |
| 審查委員 | IACUC 計畫審查 |
| IACUC 主席 | 主導審查決策 |
| 執行秘書 | 行政流程管理 |
| 試驗工作人員 | 執行實驗、記錄數據 |
| 倉庫管理員 | 進銷存系統操作 |
| 委託人 | 查看委託計畫與動物紀錄 |

### 8.3 系統設定

- 系統名稱、站台設定。
- 密碼策略設定。
- Email（SMTP）設定。

### 8.4 操作日誌 / 安全審計

- **操作日誌**：查看所有使用者的操作紀錄。
- **安全審計**：稽核日誌具有 HMAC 雜湊鏈完整性保護，確保紀錄不可竄改。

### 8.5 通知路由設定

設定各類事件的通知對象與通知管道（站內通知 / Email）。

### 8.6 藥物選單管理

管理觀察紀錄中可選的治療藥物清單與預設劑量。

## 9. 常見問題 (FAQ)

### Q: 登入後看不到某些選單？
A: 系統根據角色顯示功能選單。如需存取特定功能，請聯繫管理員指派對應角色。

### Q: 如何切換語言？
A: 點擊頂部工具列的語言選擇下拉選單，可在繁體中文與 English 之間切換。

### Q: 側邊欄選單順序可以調整嗎？
A: 可以。點擊側邊欄底部的「編輯」按鈕進入編輯模式，拖曳選單項目調整順序，點擊「完成」儲存。

### Q: 上傳檔案有什麼限制？
A: 依功能不同，一般限制為 10-30 MB，支援格式包含 PDF、Word、Excel、JPG、PNG。拖曳檔案至上傳區域即可上傳。

### Q: 忘記密碼怎麼辦？
A: 在登入頁點擊「忘記密碼」，輸入 Email 即可收到重設連結。如無法收到信件，請聯繫管理員手動重設。

### Q: 通知功能如何運作？
A: 系統會在右上角鈴鐺圖示顯示未讀通知數量。點擊鈴鐺可查看通知列表，點擊通知可跳轉至相關頁面。可點擊「全部已讀」清除未讀標記。

### Q: 電子簽章如何操作？
A: 在需要簽章的紀錄（如犧牲、觀察）頁面，點擊「簽章」按鈕，輸入登入密碼確認身分後完成簽章。已簽章的紀錄不可再修改。
