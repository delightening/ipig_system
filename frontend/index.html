豬博士 iPig 系統 — 整合型實驗動物管理平台

系統概覽

豬博士 iPig 為 GLP (Good Laboratory Practice) 合規的整合型實驗動物管理平台， 採用統一入口門戶架構，包含 AUP 計畫書審查、iPig ERP 進銷存、實驗動物醫療紀錄、 人事管理、通知與設施管理共六大子系統。

AI Agent 整合

本系統提供以下機器可讀介面，供 AI agent 與自動化工具整合：

  • A2A Agent Card — 系統能力與端點宣告
  • MCP Discovery — Model Context Protocol 端點資訊
  • WebMCP Manifest — 頁面層工具宣告
  • OpenAPI 3.0 Specification — 完整 REST API 規格
  • llms.txt — LLM 友善的系統說明
