NFC 名片寫入工具

將電子名片 URL 寫入 NFC 標籤

此瀏覽器不支援 Web NFC

Web NFC API 目前僅支援 Android 裝置的 Chrome 瀏覽器（89 版以上）。
請使用 Android 手機的 Chrome 開啟此頁面。

iOS 裝置可使用第三方 NFC 寫入 App（如 NFC Tools）手動輸入以下 URL。

寫入類型

NFC 掃描後將開啟此網址。預設為當前名片頁面。

寫入 NFC 標籤

  • 1
    填寫上方的名片資訊或確認 URL
  • 2
    點擊下方 「開始寫入」 按鈕
  • 3
    將 NFC 標籤靠近手機背面感應區域
  • 4
    等待寫入完成提示

驗證 NFC 標籤

寫入完成後，可讀取 NFC 標籤驗證內容是否正確。
